How to Define a Table
You define a table with the <table> element.

<tr>
"Table row" is used to manipulate the content of the rows of the table.

<th>
Is used to manipulate the content of a specific cell that is a header.

<td>
"Table data" is used to manipulate the contents of each cell of the table that is not the header.

<thead>
"Table head" is used to manipulate the contents of the table headers in relation to the columns.

<tfoot>
"Table footer" is used to manipulate the contents of the footer in the table.

<caption>
Is used as the table title.

Colspan vs. Rowspan
Column Span is used to merge the table cells across columns. Rowspan is used to merge the table cells across rows.

Cell Spacing vs. Padding
Cell spacing refers to the space between cells.Cell Padding refers to the space inside the cells.

Table Borders with CSS
You can define a table border using css with the "border:" tag under the specified region of table{}, td{}, or th{}. To control this border width you have to define the amount of pixels. So after "border:" you can type the number and then "px."
